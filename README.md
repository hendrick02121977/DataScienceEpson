## 📋 Course Structure

### **Day 1: Introduction & Python Basics** (4 hours)
**Learning Objectives:**
- Understand Industry 4.0 and Epson's manufacturing context
- Master Python basics (variables, data types, control flow)
- Work with NumPy arrays and Pandas DataFrames
- Analyze production sensor data

**Key Topics:**
- Python fundamentals
- Data types and structures
- NumPy operations
- Pandas data manipulation
- Case study: Production sensor analysis

---

### **Day 2: Statistics, Probability & Visualization** (4 hours)
**Learning Objectives:**
- Calculate descriptive statistics
- Understand probability distributions
- Create effective data visualizations
- Interpret statistical findings

**Key Topics:**
- Descriptive statistics (mean, median, mode, std dev)
- Normal and Binomial distributions
- Matplotlib visualization
- Seaborn statistical plots
- Case study: Epson production trends

---

### **Day 3: Data Integration & EDA Mini Project** (4 hours)
**Learning Objectives:**
- Perform exploratory data analysis (EDA)
- Integrate data from multiple sources
- Identify data quality issues
- Create comprehensive analysis reports

**Key Topics:**
- Data loading and inspection
- Data cleaning techniques
- MES/ERP/IoT system integration
- Statistical summary analysis
- **Mini Project:** Epson Production Quality Analysis

---

### **Day 4: Data Wrangling & SQL Integration** (4 hours)
**Learning Objectives:**
- Handle missing values and outliers
- Query production databases with SQL
- Merge data from multiple sources
- Extract actionable insights

**Key Topics:**
- Missing value strategies
- Outlier detection (IQR method)
- SQL database connections
- Query writing
- Data merging techniques
- **Case Study:** Production system integration
