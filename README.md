# DataScienceEpson

## Stage 1: Fundamental Data Science & Analytics

A 4-day hands-on training curriculum for Epson, covering Python-based data science, statistics, visualization, and system integration. All notebooks are designed to run in **Google Colab**.

---

### 📅 Course Outline

| Day | Notebook | Topics |
|-----|----------|--------|
| 1 | [Day1_Introduction_and_Basic_Python.ipynb](Day1_Introduction_and_Basic_Python.ipynb) | Industry 4.0 context, Python basics (variables, data types, flow control), NumPy, Pandas, Case Study: production sensor data analysis |
| 2 | [Day2_Statistics_Probability_and_Visualization.ipynb](Day2_Statistics_Probability_and_Visualization.ipynb) | Descriptive statistics, normal & binomial distributions, Matplotlib, Seaborn, Case Study: Epson production trends & machine quality |
| 3 | [Day3_Data_Integration_and_Mini_Project.ipynb](Day3_Data_Integration_and_Mini_Project.ipynb) | Data wrangling for MES/ERP/IoT, EDA, Mini Project: full analysis report with visualizations |
| 4 | [Day4_Data_Wrangling_and_System_Integration.ipynb](Day4_Data_Wrangling_and_System_Integration.ipynb) | Missing values & outliers, SQL for production databases, Case Study: merging sensor data with ERP data |

---

### 🚀 Getting Started in Google Colab

1. Open [Google Colab](https://colab.research.google.com/)
2. Click **File → Open notebook → GitHub**
3. Enter the repository URL: `https://github.com/hendrick02121977/DataScienceEpson`
4. Select the notebook for the day you want to run
5. Click **Runtime → Run all** to execute all cells

All required libraries (`numpy`, `pandas`, `matplotlib`, `seaborn`, `scipy`) are pre-installed in Google Colab.
